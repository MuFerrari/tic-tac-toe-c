# 🎯 Tic-Tac-Toe em C

> Implementação do clássico jogo da velha utilizando linguagem C, com foco em arquitetura modular, organização de código e boas práticas de desenvolvimento.

---

## 📌 Sobre o Projeto

Este projeto consiste na construção de um jogo da velha totalmente funcional em C, permitindo ao usuário interagir com o sistema via terminal.

O objetivo principal é aplicar conceitos fundamentais de programação, como:
- Lógica de programação
- Estrutura de dados
- Modularização de código
- Separação de responsabilidades

---

## 🧠 Funcionalidades

- Escolha do tamanho do tabuleiro
- Definição de nível de dificuldade
- Sistema de turnos entre jogadores
- Verificação automática de vitória ou empate
- Reinício de partida sem reiniciar o programa

---

## ⚙️ Tecnologias Utilizadas

- Linguagem C
- Compilador GCC
- Git & GitHub

---

## 🚀 Como Executar

### 1. Clone o repositório

```git clone https://github.com/seu-usuario/tic-tac-toe-c.git```

### 2. Acesse o diretório

```cd tic-tac-toe-c```

### 3. Compile o projeto

```gcc src/*.c -o jogo```

### 4. Execute

```./jogo```
